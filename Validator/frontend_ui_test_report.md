## 发现的问题

### BUG-001: 服务模块接口管理 404 错误

**模块**: 服务 (Service)
**严重程度**: 高
**状态**: 待修复

**问题描述**:
点击服务列表中的"接口"按钮打开接口管理弹窗时，显示"资源不存在"错误，后端 API 返回 404。

**复现步骤**:
1. 登录系统
2. 导航到"服务"模块 (/projects/services)
3. 点击任意服务的"接口"按钮

**期望行为**:
应该正常显示接口列表，允许添加/编辑/删除接口

**实际行为**:
显示 404 错误：`Failed to load resource: http://localhost:9527/api/v1/projects/5/epics/10/apis`

**截图位置**: `Builder/frontend/img/` 目录

---

### BUG-002: 任务看板拖拽功能失效

**模块**: 任务 (Task)
**严重程度**: 中
**状态**: 待修复

**问题描述**:
在看板中拖拽任务卡片从"待办"列到"开发中"列，拖拽后任务仍然停留在原列，状态未更新。

**复现步骤**:
1. 登录系统
2. 导航到"任务看板"模块 (/tasks)
3. 拖拽任意任务卡片到另一列

**期望行为**:
任务卡片应该移动到新列，后端状态应该更新

**实际行为**:
任务卡片视觉上似乎移动了，但页面刷新后仍在原列。控制台显示 "Draggable item task-1 was dropped over droppable area task-1"，表明放置区域识别有问题。

---

### WARNING-001: Ant Design 组件废弃属性警告

**模块**: 全局
**严重程度**: 低
**状态**: 建议优化

**问题描述**:
控制台出现多个 Ant Design 组件的废弃属性警告：

```
- [antd: Drawer] `width` is deprecated. Please use `size` instead.
- [antd: Modal] `bodyStyle` is deprecated. Please use `styles.body` instead.
- [antd: Card] `bodyStyle` is deprecated. Please use `styles.body` instead.
- [antd: Select] `dropdownClassName` is deprecated. Please use `classNames.popup.root` instead.
- [antd: Spin] `tip` is deprecated. Please use `description` instead.
- [antd: DatePicker] `dropdownClassName` is deprecated.
```

**影响**:
不影响功能，但建议在后续版本中更新以保持一致性。
